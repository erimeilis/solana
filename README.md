## Why This Project Exists

AI is not just a helper for writing code faster. And people who say it makes you dumb? They're not right.

AI can actually help you **learn** new stacks, modern technologies, and complex concepts. The aim of this project is to prove it.

I'm experienced in TypeScript. I've used Rust several times. But I know nothing about how blockchain is actually built under the hood. I want to enrich myself with this knowledge.

Let's see if AI can guide me through this path.

---

## The Experiment

This repository documents my journey learning Solana blockchain development from scratch, guided by Claude AI. Every step, every project, every mistake — all transparent.

**Starting Point:**
- TypeScript: experienced
- Rust: familiar
- Blockchain: zero

**Goal:** Build real Solana applications and understand the fundamentals.
